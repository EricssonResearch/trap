38  /**
39   * TrapState defines the set of states that TrapEndpoints can have. Uses a class as opposed to an enum in order to allow
40   * compatibility with Java versions that lack enums. In all other aspects, acts as an enum.
41   * 
42   * @author Vladimir Katardjiev
43   */
44  public enum TrapState
45  {
46      
47      /**
48       * The Trap Endpoint is closed. Client and Listener Trap Endpoints can be opened from this state; any other endpoint
49       * cannot and this is its final state.
50       */
51      CLOSED,
52      
53      /**
54       * The endpoint is in the process of establishing a connection with a remote endpoint.
55       */
56      OPENING,
57      
58      /**
59       * The Trap Session is open and, to the best of the Endpoint's knowledge, underlying transports are available. The
60       * application can use this Endpoint to send data freely.
61       */
62      OPEN,
63      
64      /**
65       * When the TrapState is SLEEPING, the TrapEndpoint does not currently have an active underlying transport to the
66       * other endpoint. Within its configuration, however, it has reason to believe that this session can be resumed.
67       * Sleeping will occur in one of the following cases:
68       * <ul>
69       * <li>The trap endpoint has lost all current transports to the other trap endpoint, but believes it can
70       * re-establish them. If it is a server endpoint, it will attempt to wake up the client and/or wait for the client
71       * to reconnect within the given time intervals in its configuration.
72       * <li>The Trap Endpoint has negotiated a connectionless state with the remote peer to save power. The endpoint will
73       * attempt to re-establish the connection once there is any data to be sent or received, but until then remain idle.
74       * </ul>
75       * While SLEEPING, the application can still use the TrapEndpoint's API and queue messages for sending. Those
76       * messages will be sent once the Endpoint is OPEN.
77       */
78      SLEEPING,
79      
80      /**
81       * An error caused the Trap Endpoint to close abnormally. This endpoint can no longer be re-used, but it can be
82       * queried to ascertain why it failed.
83       */
84      ERROR,
85      
86      /**
87       * The Trap Endpoint is attempting to close the Trap session.
88       */
89      CLOSING;
90  }
