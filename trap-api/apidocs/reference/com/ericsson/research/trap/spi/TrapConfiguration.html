com.ericsson.research.trap.spi.TrapConfiguration

Class Overview

The TrapConfiguration interface provides a specification on how to interact with a runtime (deserialized) version of TrapConfiguration format.

This interface is essentially a rename of the Trap Utils Configuration interface. The Configuration interface is reused across multiple projects, so namespacing it like this allows for easier imports in the same file.
