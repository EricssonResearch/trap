package

com.ericsson.research.trap

Classes | Description

The main API package for Trap. This is the primary entry point for writing applications that use Trap. Most of the entries herein are interfaces to objects that should be instantiated using the factory.

TrapFactory is the main factory class and is used to create new instances. To create a client, use createClient(String, boolean). To create a server (=listener), use createListener(String).

In some containers, automatic transport detection will not work. For them, before instantiating a server or client, the available transports should be added using addTransportClass(Class). This limitation is present everywhere that packages cannot be scanned by reflection.
