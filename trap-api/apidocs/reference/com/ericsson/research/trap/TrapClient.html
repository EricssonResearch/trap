TrapClient

com.ericsson.research.trap.TrapClient

Class Overview

A TrapClient is a TrapEndpoint capable of opening an outgoing connection, commonly to a TrapListener. It is able to reconnect transports at will, unlike a ServerTrapEndpoint. Create a TrapClient using createClient(String, boolean) as follows. 

 TrapClient c = TrapFactory.createClient("http://trap.server.com/");
 c.setDelegate(...)
 c.open();

TrapClient should have setDelegate(TrapDelegate, boolean) called in order to properly work. The delegate will handle state changes and incoming messages. It is recommended the delegate is set before open; doing it later may lose events.

Public Methods

public abstract void open ()

Instructs the TrapClient to begin outgoing connection attempts. This will make exactly one attempt to connect on each available transport. If any connection attempt succeeds, the entire endpoint will move to OPEN, else it will move to ERROR

Throws
TrapException If a fatal configuration error prevents the endpoint from opening. Generally this would be no available/configured transports.
