com.ericsson.research.trap.delegates.TrapEndpointDelegate

Class Overview

Convenience interface for implemting a delegate for OnData, OnStateChange and OnFailedSending . If the delegate does not need all three, it is recommended to implement the specific delegate methods instead. This interface is retained for backwards compatibility.

  • trapData(byte[], int, TrapEndpoint, Object) is called when data is received.
  • trapStateChange(TrapState, TrapState, TrapEndpoint, Object) is called when the connection state changes.
  • trapFailedSending(Collection, TrapEndpoint, Object) is called when the endpoint closes with messages left in the output buffer.
