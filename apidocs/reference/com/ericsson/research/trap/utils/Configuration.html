Class Overview

A generic configuration interface, capable of serialisation, deserialisation and some primitive operations. The supplied implementation is capable of running on virtually any JVM, providing rudimentary configuration possibilities.

public abstract Configuration getChildOptions (String optionsPrefix, boolean cutPrefixes, boolean updateParent)

Fetches a set of configuration prefixes as a Configuration instance. This instance can be used to iterate, list or modify the configuration as needed.

Parameters
optionsPrefix The prefix to extract the config for.
cutPrefixes Whether to remove the prefixes from the returned configuration
updateParent If set to true, the configuration returned will update the parent configuration if any of the child's options are altered. In essence, what this means is that the child Configuration will become a view of the parent containing only a subset of the parent's keys, but with the values backed by the parent.
Returns
  • A forked configuration

public abstract Set<String> getKeys ()

Returns the set of keys that are contained in this configuration at or after the call to getKeys() is made, but before it returns. The exact set of keys is an immutable snapshot (and thus not prone to ConcurrentModificationException), but is a complete clone of the key set each time this method is called.

  • The set of keys in this configuration

public abstract Map<String, String> getOptions (String optionsPrefix, boolean cutPrefixes)

Fetches a subset of the available options of this configuration as a live map. The options are sorted based on a certain prefix, which may optionally be removed from the map.

Any changes to the returned map will be propagated back to the parent configuration. Any changes to the parent configuration will not be reflected in the child.

Parameters
optionsPrefix The prefix string containing the options to include in the map.
cutPrefixes true to remove optionsPrefix from the keys in the returned map, false otherwise.
Returns
  • The requested subset of options, in the form of a map.
